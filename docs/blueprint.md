# **App Name**: TaskFlow

## Core Features:

- Todo Creation: Allow users to create todo items with a label, due date, priority, and assigned state.
- Customizable States: Enable users to define custom states (e.g., todo, ongoing, completed, shelved) in their profile settings.
- Todo Filtering: Allow filtering of todo items based on the label, date, state, and priority.
- Todo Editing: Enable users to edit the details of existing todo items.
- AI-Powered Smart Suggestions: Suggest the state of the to-do as a tool.

## Style Guidelines:

- Primary color: Deep purple (#6750A4) to convey focus and productivity.
- Background color: Very light purple (#F3F0F7) to create a calming, uncluttered workspace.
- Accent color: Blue (#5FA8D3) for actionable items and key interactions.
- Body and headline font: 'Inter' (sans-serif) for a modern, machined, objective, neutral feel suitable for headlines or body text
- Use simple, geometric icons to represent different states and priorities.
- Clean, minimalist layout with clear visual hierarchy to enhance focus.
- Subtle animations to transition between states and highlight task updates.
